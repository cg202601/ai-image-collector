# AI Image Collector 用户手册

AI Image Collector 是一款用于收集和管理 AI 生成图片素材的 Chrome 扩展。它可以把图片保存到你自己的 Google Drive，并把提示词、来源链接、软件名称、分类、备注等信息记录到你自己的 Google Sheets，方便后续检索、复用和整理。

这份手册面向普通使用者，不涉及开发细节。

## 一、适合谁使用

适合这些场景：

- 经常在 AI 生图网站上收集图片和提示词。
- 想把图片、提示词、来源、软件、分类统一整理起来。
- 希望后续能按分类、标签、软件或关键词查找历史素材。
- 希望把自己的 AI 图片资料沉淀成可复用的词库。

不适合这些场景：

- 只想临时下载一两张图片，不需要长期管理。
- 不使用 Google Sheets 或 Google Drive。
- 不希望自己配置 GAS、表格和网盘。

## 二、你需要准备什么

使用前需要准备：

- Chrome 或 Chromium 内核浏览器。
- 一个 Google 账号。
- 一个 Google Sheets 表格，用来保存图片记录。
- 一个 Google Drive 文件夹，用来保存上传的图片。
- 一个已部署好的 Google Apps Script Web App 地址。

如果你拿到的是组织已经配置好的版本，可能只需要填写管理员提供的 GAS 地址、表格 ID 和网盘文件夹 ID。

## 三、安装扩展

1. 下载官方发布的压缩包。
2. 解压压缩包。
3. 打开 Chrome，进入 `chrome://extensions/`。
4. 打开右上角的“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择解压后的扩展文件夹。
7. 建议把扩展固定到浏览器工具栏，方便进入设置。

注意：请优先使用组织审核通过的 Release 包，不建议使用别人私下发送的未验证压缩包。

## 四、首次配置

点击浏览器工具栏里的扩展图标，进入设置页面。

需要填写的主要内容：

| 配置项 | 说明 |
| --- | --- |
| GAS Web App URL | Google Apps Script 部署后的 Web App 地址 |
| Google Sheets ID | 表格 URL 中 `/d/` 和 `/edit` 之间的 ID |
| Google Drive Folder ID | 网盘文件夹 URL 中 `/folders/` 后面的 ID |
| 默认人名 | 写入表格的人名，可填自己的名字或团队成员名 |
| 网站版图库地址 | 可选，如果团队部署了展示站，可填写展示站地址 |

填写完成后点击保存。建议先提交一张测试图片，确认表格和网盘都能正常写入。

## 五、收集图片

扩展支持几种常用收集方式。

### 方式一：右键收集

1. 在网页中找到想保存的图片。
2. 右键图片。
3. 选择扩展提供的收集菜单。
4. 在弹出的收集面板中填写提示词、软件、分类和备注。
5. 点击提交。

### 方式二：粘贴图片

1. 复制一张图片，或先截图到剪贴板。
2. 打开收集面板。
3. 在图片区域粘贴。
4. 填写信息后提交。

### 方式三：拖拽图片

1. 打开收集面板。
2. 把图片拖进面板。
3. 填写信息后提交。

## 六、填写信息建议

为了后期好找，建议至少填写这些内容：

- `软件`：例如 Midjourney、Nano banana2、Flow 等。
- `分类`：例如 MV素材、圣经故事、道具图、人像、场景等。
- `提示词`：尽量保留完整提示词。
- `备注`：写你以后能看懂的补充说明，例如“光影好”“构图参考”“适合封面”。
- `来源链接`：扩展能识别时会自动记录，必要时也可以手动补充。

小建议：分类不要太随意。前期可以少一点，后面逐步稳定下来。

## 七、词库中心

词库中心用于查看和复用历史提示词、模板和收藏内容。

常见用途：

- 搜索历史提示词。
- 查看模板提示词。
- 收藏常用提示词。
- 查看高频提示词。
- 通过预览图快速判断提示词效果。
- 复制提示词继续使用。

如果你发现词库没有加载，通常是 GAS 地址、表格 ID 或权限配置有问题。

## 八、模板和历史

扩展会把历史记录和模板统一展示到词库中心。

- `历史`：来自你以前保存到表格里的记录。
- `模板`：来自表格中的 Templates 工作表。
- `收藏`：你在扩展里标记收藏的提示词。
- `高频`：根据使用次数、复制次数等信息判断更常用的内容。

模板适合放稳定可复用的提示词结构，历史更适合临时查找以前保存过的素材。

## 九、标签和分类

如果表格中配置了 TagCloud，扩展会读取云端标签词库。

标签适合用来快速拼接或筛选提示词，例如：

- 时间：清晨、黄昏、夜晚。
- 风格：写实、电影感、复古。
- 镜头：特写、广角、俯拍。
- 光影：逆光、柔光、体积光。

标签不是必须项，但用得越久，越能提升检索效率。

## 十、配置云同步

扩展支持配置云端上传和下载。

适合这些情况：

- 换电脑后恢复扩展配置。
- 团队成员使用同一套软件列表、分类和标签。
- 防止本地浏览器配置丢失。

建议在配置稳定后上传一次云端配置。下载云端配置前，确认当前本地配置是否需要保留，避免覆盖。

## 十一、失败重试

如果提交时网络异常、GAS 暂时失败或 Google 服务响应慢，扩展会尽量保留失败任务。

你可以稍后点击重试，不必重新填写全部信息。

如果连续失败，优先检查：

- GAS URL 是否正确。
- 表格 ID 是否正确。
- GAS 是否重新部署过最新版本。
- Google 账号是否有表格和网盘权限。

## 十二、网站版图库

如果团队配置了网站版图库，可以在扩展设置里填写网站地址。

网站版通常用于：

- 大图浏览历史作品。
- 点赞、收藏和排序。
- 按标签检索。
- 查看提示词和备注。
- 复制提示词。

网站版是扩展的可选配套工具。没有网站版时，扩展本体仍然可以正常收集图片。

## 十三、常见问题

### 1. 提交后表格没有新增记录

检查 GAS URL 和 Google Sheets ID 是否填写正确。也可以打开 GAS 部署地址，看是否能正常访问。

### 2. 图片上传了，但表格里的预览不显示

通常和 Google Drive 权限或图片链接有关。确认目标文件夹可被 GAS 写入，并且图片文件没有被权限策略阻止访问。

### 3. 词库中心加载失败

检查表格 ID、GAS 权限和 GAS 是否是最新部署版本。修改 GAS 代码后，需要在 Apps Script 中重新部署新版本。

### 4. 快捷键不好用

有些网页会拦截快捷键。可以使用扩展界面里的入口按钮，或通过工具栏打开设置和相关页面。

### 5. 云上传或云下载失败

确认 GAS 中包含配置同步功能，并且当前部署版本是最新的。

## 十四、使用习惯建议

- 每次保存时尽量写清楚分类和软件。
- 重要提示词及时收藏。
- 高质量提示词可以整理成模板。
- 不要把真实密钥、账号密码或敏感项目资料写进提示词。
- 定期检查表格和网盘是否正常增长。
- 大版本更新前，可以先备份当前扩展配置。

## 十五、数据保存在哪里

扩展本身不提供独立服务器。

你的主要数据保存位置：

- 图片：你的 Google Drive。
- 记录：你的 Google Sheets。
- 本地配置：浏览器扩展存储。
- 云端配置：你的 Google Sheets 配置表。

也就是说，数据主要由你自己的 Google 账号资源保存和管理。

## 十六、给团队成员的简短说明

如果你只是使用者，可以记住三件事：

1. 看到好图就收集。
2. 分类、软件、提示词尽量写清楚。
3. 找历史素材时去词库中心或网站版图库。

这套工具的核心价值不是“保存一张图”，而是把图片、提示词和灵感长期整理成可搜索、可复用的素材库。
